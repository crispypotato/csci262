This assignment code is written and compiled on IntelliJ IDEA Community Edition 2022.2.2
Java version: JDK 18.0.2.1

To run the program:
1) Compile in terminal (javac IDS.java)
2) Run program along with inclusion of events file, statistics file and number of days to generate
   eg. events file is Events.txt, statistics file is Stats.txt, number of days = 5
   --> enter (java IDS.java Events.txt Stats.txt 5)
3) After going through the activity and analysis engine, the program will prompt for a new statistics file. Enter the file in the format ____.txt (eg Stats2.txt). Afterwards, enter the new number of days when prompted.

All files mentioned above are included in the zip file. To read in other files, copy them into the same folder.
